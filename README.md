## 🗂️ 프로젝트 소개
### 프로젝트 명 - 너 내 동료가 돼라
- 프로젝트 팀원을 구하는 사이트
    - 사용자는 자신의 기술, 경험, 관심사 등을 포함한 프로필을 작성하여 자신의 기술과 맞는 프로젝트를 찾을 수 있습니다.
    - 프로젝트의 목적, 필요한 기술 등을 설명하여 프로젝트를 게시하고 필요한 기술과 경험이 있는 팀원을 구할 수 있습니다.
